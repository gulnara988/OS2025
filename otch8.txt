Проверка записи информации в файл и чтения информацияя из файла 
Для открытия файла используется системный вызов open()
открытие для записи:fd=open("myfile", O_WRONLY | O_CREAT , 0666);
MYFILE -относительный путь к файлу ,O_WRONLY | O_CREAT - флаги для открытия  только для записи и создания файла , если он отсутствует, 0666 - права доступа к файлу (чтение/запись для всех пользователей)
Открытие для чтения : fd=open("myfile",O_RDONLY)
Запись информации производится при помощи системного вызова write():
size= write(fd, string, 20);
первый параметр - файловый дискриптор, определяющий средство связи, через,  которое пишется информация
Второй параметр - источник информации (символный массив)
Третий параметр - количество байт , которое планирует запсиать
size - значение количсетва фактически записанных файл байт
Чтениеиформации производится прир помощи системного вызоваread();
size= read(fd, string, 20);
fd - файловый дискриптор
string - приемник информации , куда записывается байтовый поток
20 - колчество байт, которые требуется прочитать
size - количество фактического прочитонных байт

Системного вызов close() закрывает файл с дискриптором fd
